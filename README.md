# Move-It Website

A moving services website built with Bootstrap 5.

View Here: https://markyyy2856.github.io/Move-It-Website/

## About

Move-It is a startup providing moving services, offering professional and personalized moving experiences.

## Bootstrap Features

This site uses Bootstrap 5 for:

- **Responsive navbar** with collapsible menu
- **Grid layout** for flexible page structure
- **Buttons and forms** for user interactions
- **Spacing utilities** for consistent padding and margins
- **Typography classes** for styled headings and text
- **Flexbox utilities** for element alignment

## Getting Started

1. Open `index.html` in a browser
2. Navigate through the site using the responsive navbar
3. View on different screen sizes to see the responsive design in action
   Note: This is only for visual representation, there's no button functionalities

## Bootstrap CDN

Bootstrap 5.3.0-alpha2 is loaded via CDN from jsDelivr.

To enable dark mode, add `data-bs-theme="dark"` to the `<html>` tag.
